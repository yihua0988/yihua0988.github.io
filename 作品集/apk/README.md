## 🎮 核心玩法

* **過關獎勵：** 只要題目全部答對，系統就會自動跳轉解鎖我的個人作品集網站 ([https://yihua0988.github.io/](https://yihua0988.github.io/))。
* **選錯懲罰：** 如果中途不小心（或故意）答錯，會直接強制跳出全螢幕廣告。這不是普通的廣告，你**必須乖乖完整看完**，系統才會放行讓你重新挑戰。

## 🛠️ 開發技術與實作細節

雖然是個簡單的互動 App，但在防呆跟體驗上做了一些防護機制：

* **防作弊與影音鎖定：** 
  為了防止偷吃步，我用計時器綁定了影片進度，把快轉和暫停功能都拔掉了。同時在底層邏輯鎖死手機的「實體返回鍵」跟「螢幕旋轉」，想卡 Bug 跳過懲罰是不可能的。

* **硬體級強制音量：** 
  懲罰就是要徹底！我導入了擴充套件來控制硬體，只要進入懲罰畫面，App 會無視你手機原本的靜音設定，直接強制開啟媒體音量。

* **彈性 UI 排版：** 
  為了適應不同尺寸的手機螢幕，我沒有把長寬數值寫死，而是塞入「隱形標籤」來當作畫面的彈性骨架。這樣不管是大螢幕還是小手機，介面都不會跑版。

* **App內建存檔機制：** 
  首頁接了 TinyDB 來做本地端存檔，結合起始值邏輯，App 會自動記錄使用者的回訪次數。就算你把 App 整個滑掉重開，紀錄也不會歸零。
